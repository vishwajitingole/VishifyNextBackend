Next.js Vercel-ready API (converted from Express)
================================================

Instructions:
1. Copy the `.env` variables into Vercel project settings (do NOT commit .env).
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`

.env keys (example):
MONGO_URI=
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
EMAIL_HOST=
EMAIL_PORT=
EMAIL_USER=
EMAIL_PASS=
NODE_ENV=production
PORT=3000
